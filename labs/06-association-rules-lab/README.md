# Lab 6: Association Rules - Market Basket Analysis

## Objectives

In this lab, you will:
- Perform market basket analysis on retail transactions
- Apply Apriori algorithm
- Use FP-Growth for efficiency
- Generate and evaluate association rules
- Make business recommendations
- Build a recommendation system

## Dataset

**Online Retail Dataset**
- Transaction data from UK retailer
- 500K+ transactions
- Product descriptions, quantities, prices
- Customer IDs and countries

## Lab Structure

- **Part 1**: Data Preparation (30 min)
- **Part 2**: Apriori Algorithm (40 min)
- **Part 3**: FP-Growth (30 min)
- **Part 4**: Rule Generation & Evaluation (30 min)
- **Part 5**: Business Applications (20 min)

## Grading

- Part 1: 20 points
- Part 2: 25 points
- Part 3: 20 points
- Part 4: 20 points
- Part 5: 15 points
- **Total**: 100 points

- Understand association rule mining
- Apply support-confidence framework
- Use lift for rule quality
