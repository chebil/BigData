# Lab 5: Regression - House Price Prediction

## Objectives

In this lab, you will:
- Build regression models to predict house prices
- Perform feature engineering
- Compare Linear, Ridge, Lasso, and Polynomial regression
- Diagnose and fix regression problems
- Validate models using cross-validation
- Deploy a production price prediction system

## Dataset

**Boston Housing Dataset** (or California Housing)
- Features: location, size, age, amenities
- Target: House price (median value)
- ~500 samples with 13 features

## Lab Structure

- **Part 1**: Data Exploration & Feature Engineering (30 min)
- **Part 2**: Linear Regression (30 min)
- **Part 3**: Regularization (Ridge, Lasso, Elastic Net) (40 min)
- **Part 4**: Polynomial Regression (30 min)
- **Part 5**: Model Comparison & Deployment (20 min)

## Grading

- Part 1: 20 points
- Part 2: 20 points
- Part 3: 25 points
- Part 4: 20 points
- Part 5: 15 points
- **Total**: 100 points

## Learning Outcomes

- Master regression fundamentals
- Handle multicollinearity
- Apply regularization techniques
- Validate model assumptions
- Build production pipelines
