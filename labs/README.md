# Labs Overview

Comprehensive hands-on laboratories for Big Data Analytics.

## Available Labs

1. **Lab 0**: Environment Setup
2. **Lab 2**: Statistics & Hypothesis Testing
3. **Lab 3**: Clustering - Customer Segmentation
4. **Lab 4**: Classification - Fraud Detection
5. **Lab 5**: Regression - House Price Prediction
6. **Lab 6**: Association Rules - Market Basket
7. **Lab 7**: Model Selection & Tuning
8. **Lab 8**: Time Series Forecasting
9. **Lab 9**: Text Analytics - Sentiment Analysis
10. **Lab 10**: Spark & Distributed Computing
11. **Capstone**: End-to-End Project

## Lab Structure

Each lab includes:
- README with objectives
- Exercise notebooks
- Datasets
- Grading rubric
- Solution guides

## Grading

Total: 500 points (50% of final grade)
- Each lab: 100 points
- Bonus challenges: Up to 10 points each
