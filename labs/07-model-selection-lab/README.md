# Lab 7: Model Selection & Hyperparameter Tuning

## Objectives

In this lab, you will:
- Implement various cross-validation strategies
- Perform hyperparameter tuning with Grid Search
- Use Random Search and Bayesian Optimization
- Compare multiple models statistically
- Build automated ML pipelines
- Select the best model for deployment

## Dataset

**Multiple Datasets**:
1. Classification: Breast Cancer Wisconsin
2. Regression: Boston Housing
3. Time Series: Airline Passengers

## Lab Structure

- **Part 1**: Cross-Validation Strategies (40 min)
- **Part 2**: Grid Search & Random Search (40 min)
- **Part 3**: Bayesian Optimization (30 min)
- **Part 4**: Model Comparison (30 min)
- **Part 5**: Automated ML Pipelines (30 min)
