# Lab 3: Clustering - Customer Segmentation

## Objectives

In this lab, you will:
- Apply K-Means clustering to real customer data
- Perform hierarchical clustering
- Use DBSCAN for density-based clustering
- Determine optimal number of clusters
- Evaluate clustering quality
- Build a customer segmentation system

## Dataset

**Mall Customer Segmentation Dataset**
- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

## Lab Structure

- **Part 1**: Data Exploration (20 minutes)
- **Part 2**: K-Means Clustering (40 minutes)
- **Part 3**: Hierarchical Clustering (30 minutes)
- **Part 4**: DBSCAN (30 minutes)
- **Part 5**: Business Application (30 minutes)

## Grading

- Part 1: 15 points
- Part 2: 25 points
- Part 3: 20 points
- Part 4: 20 points
- Part 5: 20 points
- **Total**: 100 points

## Learning Outcomes

By the end of this lab, you will be able to:
1. Preprocess data for clustering
2. Apply multiple clustering algorithms
3. Determine optimal clusters using elbow method and silhouette analysis
4. Interpret and visualize clusters
5. Make business recommendations based on segments
