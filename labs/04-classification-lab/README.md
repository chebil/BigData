# Lab 4: Classification - Fraud Detection

## Objectives

In this lab, you will:
- Build a credit card fraud detection system
- Handle imbalanced datasets
- Compare multiple classification algorithms
- Optimize model performance
- Evaluate using appropriate metrics
- Deploy a production classifier

## Dataset

**Credit Card Fraud Detection Dataset**
- Features: V1-V28 (PCA transformed)
- Time: Seconds since first transaction
- Amount: Transaction amount
- Class: 0 (genuine), 1 (fraud)
- **Highly imbalanced**: ~0.17% fraud

## Lab Structure

- **Part 1**: Data Exploration & Preparation (30 minutes)
- **Part 2**: Logistic Regression (30 minutes)
- **Part 3**: Decision Trees & Random Forests (40 minutes)
- **Part 4**: Model Comparison & Optimization (30 minutes)
- **Part 5**: Production Deployment (20 minutes)
