# Course Syllabus

## Course Information

**Course Title**: Big Data Analytics  
**Instructor**: Dr. Chebil Khalil  
**Credits**: 3  
**Prerequisites**: Basic programming, introductory statistics  
**Format**: Self-paced with hands-on labs

## Course Description

This course provides comprehensive coverage of data science and big data analytics methodologies, tools, and techniques. Students will learn the complete data analytics lifecycle, from data acquisition and preparation through model building, evaluation, and deployment using modern Python-based tools and Apache Spark.

## Learning Outcomes

Upon successful completion of this course, students will be able to:

1. **Apply the Data Analytics Lifecycle** to real-world business problems
2. **Perform exploratory data analysis** using Python, Pandas, and visualization libraries
3. **Implement machine learning algorithms** including clustering, classification, regression, and text analytics
4. **Process large-scale datasets** using Apache Spark and distributed computing
5. **Communicate analytical findings** effectively to technical and non-technical audiences
6. **Deploy predictive models** in production environments

## Course Topics

### Part I: Foundations (Weeks 1-3)
- Introduction to Big Data and Analytics
- Six-Phase Data Analytics Lifecycle
- Statistical Foundations with Python
- Data Visualization and Exploratory Analysis

### Part II: Machine Learning Methods (Weeks 4-9)
- Clustering Analysis (K-means, Hierarchical, DBSCAN)
- Association Rules Mining
- Regression Analysis (Linear, Logistic, Regularization)
- Classification Methods (Decision Trees, SVM, Random Forests)
- Time Series Analysis and Forecasting
- Text Analytics and NLP

### Part III: Big Data Technologies (Weeks 10-12)
- Distributed Computing with Hadoop and Spark
- Advanced Topics (Deep Learning, Deployment)
- Capstone Project
