## FAQ

**Q: I've never programmed before. Can I take this course?**
A: This course assumes basic programming knowledge. We recommend completing a Python fundamentals course first.

**Q: I know R but not Python. Is that okay?**
A: Yes! Python and R are similar. Review our Python cheatsheet and you'll catch up quickly.

**Q: Do I need a powerful computer?**
A: 8GB RAM is sufficient for most exercises. For Spark labs, 16GB is better but not required.

**Q: Can I use Windows/Mac/Linux?**
A: Yes! Docker works on all platforms. Course is OS-agnostic.

**Q: How much time should I budget per week?**
A: Plan for 10-15 hours: 4-6 for reading, 6-8 for labs, plus time for quizzes.
